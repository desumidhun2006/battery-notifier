# Android

## Install (One Click)

1. Go to [Releases](https://github.com/desumidhun2006/battery-notifier/releases)
2. Download `battery-notifier-android.apk`
3. Open the APK on your phone
4. If prompted, allow "Install from unknown sources"
5. Tap **Install**

## Use

1. Open Battery Notifier
2. Set your target percentage
3. Tap **Activate Alarm**
4. Plug in charger — alarm fires at target
5. Unplug to dismiss

## Notes

- Requires Android 5.0+
- Alarm works in background
- Allow notifications when prompted
